# Time Tracker

Toggl Track 風の個人用タイムトラッカー。バックエンド不要、ブラウザだけで完結します。

## 使い方

`index.html` をブラウザで開くだけです（ダブルクリック or `file://` で開く）。
ビルドやサーバ起動は不要です。HTML・CSS・JS は 1 ファイルにまとめてあります。
Vue 3 と VueUse のみ CDN から読み込むため、初回だけネット接続が必要です（以降はブラウザキャッシュで動きます）。

```txt
time-tracker/
├── index.html   # 画面・スタイル・ロジックを含む単一ファイル
└── README.md
```

## 画面構成

1 画面の 2 カラム構成（狭い幅では 1 カラムに折り返し）。

- **左**: エントリー（カレンダー / リスト切替）とレポート。
- **右上の ⚙ ボタン**: プロジェクトとデータ管理をスライド式ドロワーで開閉（普段は隠れています）。Esc / 背景クリックで閉じます。

## 機能

### エントリー

カレンダー表示とリスト表示を見出し右のトグルで切り替えます。**元に戻す / やり直し**（Ctrl+Z / Ctrl+Shift+Z、見出し右のボタン）に対応。

- **カレンダー表示**: Google カレンダー風の週ビュー（月曜始まり・24時間）。
  - 空き時間を**ドラッグ**して作成。単クリックでも 30 分の予定を作成。
  - ブロックを**中央ドラッグで移動**（別の曜日へも移動可）、**上下端ドラッグでリサイズ**。15 分単位でスナップ。
  - ブロックを**クリックで詳細編集**（説明・プロジェクト・タグ・開始/終了時刻・削除）。
  - ブロックにプロジェクト名とタグを表示。プロジェクト未設定はグレーで区別。
  - 「‹ / 今週 / ›」で週を移動。曜日ヘッダに**日別合計**、右上に**週合計**を表示。
- **リスト表示**: エントリを新しい順に一覧。**期間**（今日 / 今週 / 今月 / 今年 / 先週 / 先月）で絞り込み、合計時間を表示。行クリックで詳細編集。

### レポート

**期間**（今日 / 今週 / 今月 / 今年 / 先週 / 先月）と**集計軸**（プロジェクト別 / タグ別）を選んでバー表示。タグ別は 1 エントリが複数タグを持つ場合に各タグへ加算するため、割合の合計は 100% を超えることがあります。

### プロジェクト（⚙ ドロワー）

名前と色で管理。エントリに紐付け。削除すると関連エントリは「プロジェクトなし」に。

### データ管理（⚙ ドロワー）

JSON でエクスポート / インポート（バックアップ・移行用）。インポートは現在のデータを置き換えます。

## データ保存

- ブラウザの `localStorage`（キー: `time-tracker-data`）に自動保存されます。
- 同じブラウザ・同じオリジンでのみ参照できます。別端末へ移すときは JSON エクスポート/インポートを使ってください。
- ブラウザのデータを消すと記録も消えるため、定期的なエクスポートを推奨します。
