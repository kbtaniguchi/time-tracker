# Time Tracker

Toggl Track 風の個人用タイムトラッカー。バックエンド不要、ブラウザだけで完結します。

## 使い方

`index.html` をブラウザで開くだけです（ダブルクリック or `file://` で開く）。
ビルドやサーバ起動は不要です。HTML・CSS・JS は 1 ファイルにまとめてあります。
Vue 3 と VueUse のみ CDN から読み込むため、初回だけネット接続が必要です（以降はブラウザキャッシュで動きます）。

```txt
time-tracker/
├── index.html   # 画面・スタイル・ロジックを含む単一ファイル
└── README.md
```

## 機能

- **カレンダー（週表示）**: Google カレンダー風の週ビュー（月曜始まり・24時間）。
  - 空き時間を**ドラッグ**して予定（エントリ）を作成。単クリックでも30分の予定を作成。
  - ブロックを**中央ドラッグで移動**（別の曜日へも移動可）、**上下端ドラッグでリサイズ**。15分単位でスナップ。
  - ブロックを**クリックで詳細編集**（説明・プロジェクト・タグ・開始/終了時刻・削除）。
  - 「‹ / 今週 / ›」で週を移動。週合計を表示。
- **プロジェクト**: 名前と色で管理。エントリに紐付け。削除すると関連エントリは「プロジェクトなし」に。
- **レポート**: 今日 / 今週（月曜始まり）/ 全期間でプロジェクト別の合計をバー表示。
- **データ管理**: JSON でエクスポート/インポート（バックアップ・移行用）。

## データ保存

- ブラウザの `localStorage`（キー: `time-tracker-data`）に自動保存されます。
- 同じブラウザ・同じオリジンでのみ参照できます。別端末へ移すときは JSON エクスポート/インポートを使ってください。
- ブラウザのデータを消すと記録も消えるため、定期的なエクスポートを推奨します。
